# Student Grade Calculator Website

A beginner-friendly website that calculates average marks, assigns grades, and determines pass/fail status.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Student name input
- Multiple marks input
- Average calculation
- Grade assignment
- Pass/Fail status
